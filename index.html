⚠️ Nécessite Hetzner + n8n pour les vraies ventes

⚙️ Workflow n8n — à configurer quand tu as Hetzner

Tout est prêt, colle juste ta clé Etsy + ton topic ntfy

1
Hetzner + Coolify — installe n8n via Coolify (tu l'as déjà prévu)
2
Importe le workflow ci-dessous dans n8n
3
Remplace TON_API_ETSY par ta Keystring
4
Remplace TON_TOPIC_NTFY par ton topic
5
Active le workflow → les notifs arrivent automatiquement 🎉
{
  "name": "Etsy → Notif vente",
  "nodes": [
    {
      "type": "n8n-nodes-base.scheduleTrigger",
      "name": "Vérif toutes les 60s",
      "parameters": { "rule": { "interval": [{ "field": "seconds", "secondsInterval": 60 }] } }
    },
    {
      "type": "n8n-nodes-base.httpRequest",
      "name": "API Etsy — Nouvelles commandes",
      "parameters": {
        "url": "https://openapi.etsy.com/v3/application/shops/MON_SHOP_ID/receipts",
        "headers": { "x-api-key": "TON_API_ETSY" },
        "qs": { "limit": 5, "was_paid": true }
      }
    },
    {
      "type": "n8n-nodes-base.httpRequest",
      "name": "Notif ntfy.sh",
      "parameters": {
        "method": "POST",
        "url": "https://ntfy.sh/TON_TOPIC_NTFY",
        "body": "🛍️ Nouvelle vente : {{ $json.product_name }} — {{ $json.grandtotal }} € · {{ $json.country_iso }}",
        "headers": {
          "Title": "Nouvelle vente Etsy !",
          "Priority": "high",
          "Tags": "shopping"
        }
      }
    }
  ]
